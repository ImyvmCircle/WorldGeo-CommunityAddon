## 聚落名称修改

点击**聚落名称修改**将打开**辖区选择界面**。该按钮位于槽位 19，图标为命名牌。可选对象分为两类。

- **全局**用于修改聚落的总名称。
- 某个**辖区范围**用于单独修改该辖区范围的名称。

选择后，铁砧界面打开，当前名称已预填。输入新名称并取出输出物品后进入确认流程。系统会在聊天框发出带费用的互动账单。

- **[确认]** — 执行改名并扣除费用；
- **[取消]** — 取消本次操作。

确认提示在 **5 分钟**内有效。

**费用与间隔限制：**

| 修改对象 | 费用 | 间隔时间 |
|---|---|---|
| 聚落总名称 | $2,000.00 | 每个聚落 30 天（现实时间） |
| 单个辖区范围名称 | $100.00 | 每个辖区范围 30 天（现实时间） |

某个名称在过去 30 天内改动过时，点击改名按钮会立即被阻止，铁砧界面不会打开，系统会提示剩余等待天数。间隔以**名称**为单位独立计算，聚落总名称与各辖区范围名称互相独立，服务器重启后间隔继续保留。辖区范围改名后，如 A 改为 B，间隔转到新名称 B；原名称 A 的间隔解除，B 的 30 天间隔从此次改名日期起算。

> **注意：** 行政助理执行此操作需要启用**重命名领域**行政授权；所有者始终豁免授权检查。

改名确认和取消都通过聊天框按钮完成。

## 管理成员

点击**管理成员**将以管理员视图打开**成员列表菜单**，显示所有成员及其角色。

### 成员详情菜单

点击成员头颅将打开其**成员详情菜单**，行政助理可在此执行以下操作。

- **提升为行政助理 / 降级为成员** — 更改成员角色。
- **移出领域** — 踢出该成员。
- **管理行政授权** — 打开该行政助理的**行政授权菜单**，该操作面向所有者开放。

### 操作限制

| 操作 | 所有者 | 行政助理 |
|---|---|---|
| 提升 / 降级成员 | ✔ | ✔（仅限普通成员） |
| 移出成员 | ✔ | ✔（仅限普通成员） |
| 提升为行政助理 | ✔ | ✗ |
| 降级 / 移出其他行政助理 | ✔ | ✗ |
| 降级 / 移出所有者 | ✗ | ✗ |

> **注意：** 最多可设置 **3 名**行政助理。

### 行政授权

**所有者**可以为每位行政助理单独切换各项授权。在成员详情菜单中打开的**行政授权菜单**内进行配置。

所有行政授权**默认全部启用**。所有者可禁用其中任意项。所有者本人不受各项开关限制；行政助理即使拥有按钮，也会在执行时再检查授权和聚落状态。

| 授权 | 说明 |
|---|---|
| **重命名领域** | 允许行政助理修改聚落总名称及各辖区范围名称 |
| **管理成员** | 允许行政助理提升 / 降级 / 移出成员 |
| **审核新成员申请** | 允许行政助理接受或拒绝申请者 |
| **编辑内政公告** | 允许行政助理创建和删除公告 |
| **国库赠予金币** | 允许行政助理发起并接受国库赠予（向其他聚落国库赠出或代表本聚落接受） |
| **国策树规划** | 允许行政助理管理国策树（计划中） |
| **地理范围修改** | 允许行政助理修改领地形状和大小 |
| **地块设置项修改** | 允许行政助理调整领地权限和规则 |
| **管理传送点** | 允许行政助理设置和配置传送点 |
| **移民政策** | 允许行政助理更改领域移民政策 |

## 审核新成员申请

点击**审核新成员申请**将打开**待审移民列表菜单**，列出所有处于**待审移民**状态的玩家。

- 点击申请者可打开其**审核菜单**，内含**接受**和**拒绝**按钮。

### 接受

- 申请者晋升为**成员**，从这时起计入正式成员。
- 他们收到邮件通知。
- 申请者为**受邀**玩家时，加入费用从领域**国库**中扣除。国库余额不足时，审核将被阻止。
- 申请者通过**申请制**政策直接申请时，费用已在申请时由玩家支付。
- 新成员加入后，聚落在整个领域范围内设置了与默认状态不同的通行权限时，系统会尽量给新成员补上个人权限，避免普通成员被本聚落自己的全局限制误伤。PvP 和飞行不参与自动补授权。

### 拒绝

- **直接申请**的玩家会收到加入费用退款，其状态设为**被拒移民**。
- **受邀**申请者的邀请记录会被删除，且不涉及费用，因为邀请阶段未收取费用。

> **注意：** 需要**审核新成员申请**行政授权。

## 内政公告管理

点击**编辑内政公告**将以管理员视图打开**公告列表菜单**。已删除公告会单独标记，并显示在列表中。

### 创建公告

在列表菜单中点击**创建公告**，铁砧界面出现 — 输入公告内容。

- 公告立即发布给所有成员。
- 成员下次登录或打开公告列表时会看到通知。
- 已读状态按成员单独追踪。

**对应命令：** `/community announcement create <communityIdentifier> <content>`

### 删除公告

在列表中点击公告以打开详情视图，然后点击**删除**。公告采用软删除方式处理，对成员隐藏，但保留在记录中。

> **注意：** 需要**编辑内政公告**行政授权。

**对应命令：** `/community announcement delete <communityIdentifier> <announcementId>`

## 国库赠予金币

点击**国库赠予金币**将打开**赠予目标列表菜单**。该按钮位于槽位 13，图标为金锭。发起方可从列表中选择要赠予的目标社区。

### 操作流程

1. 在目标列表中点击目标聚落；
2. 在**赠予金额菜单**中选择赠予金额；
3. 系统在聊天栏发送确认请求，并附带可点击的 **[接受]**、**[拒绝]**、**[取消]** 按钮。接受和拒绝由目标聚落的领主或具备授权的行政助理点击，取消由发起聚落的领主或具备授权的行政助理点击。
4. 确认提示将在 **5 分钟**后过期。

赠予记录将作为支出计入**发起聚落**国库，并作为收入计入**目标聚落**国库，均计入各自的总资产。

> **注意：** 发起方与接受方均需具备**国库赠予金币**行政授权（领主始终豁免授权检查）。

赠予的接受、拒绝和取消都通过聊天框按钮完成。

## 移民政策

**移民政策**按钮位于槽位 28。每次点击都会在三种政策之间循环切换。

| 政策 | 图标 | 行为 |
|---|---|---|
| **公开** | 绿色羊毛 | 玩家确认并支付加入费用后立即成为成员 |
| **申请制** | 黄色羊毛 | 玩家提交申请时支付费用，成为待审移民，等待审核；拒绝时退款 |
| **仅限邀请** | 红色羊毛 | 玩家通过成员邀请加入；接受邀请后进入审核流程，费用由国库在审核通过时支付 |

> **注意：** 需要**移民政策**行政授权。
