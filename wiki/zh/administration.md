## 聚落名称修改

点击**聚落名称修改**（槽位 19，命名牌图标）将打开**辖区选择界面**，你可以选择：

- **全局** — 修改聚落的总名称；或
- 某个**辖区范围（Geoscope）** — 仅修改该辖区范围的名称。

选择后，铁砧界面出现，当前名称已预填。输入新名称并取出输出物品，进入确认流程。系统在聊天框发出含费用的互动账单：

- **[确认]** — 执行改名并扣除费用；
- **[取消]** — 取消本次操作。

确认提示在 **5 分钟**内有效。

**费用与冷却：**

| 修改对象 | 费用 | 冷却时间 |
|---|---|---|
| 聚落总名称 | $2,000.00 | 每个聚落 30 天（现实时间） |
| 单个辖区范围名称 | $100.00 | 每个辖区范围 30 天（现实时间） |

若某个名称在过去 30 天内已被修改过，该操作将被阻止，并提示剩余冷却天数。冷却对每个名称项单独计算（聚落总名称与各辖区范围名称互相独立），服务器重启后冷却依然保留。

> **注意：** 此操作需要你的行政助理角色启用**重命名领域**行政授权（所有者始终豁免授权检查）。

**对应命令：**
- `/community confirm_rename <regionId> <nameKey>` — 确认改名
- `/community cancel_rename <regionId> <nameKey>` — 取消改名

## 管理成员

点击**管理成员**将以管理员视图打开**成员列表菜单**，显示所有成员及其角色。

### 成员详情菜单

点击成员头颅将打开其**成员详情菜单**，行政助理可在此：

- **提升为行政助理 / 降级为成员** — 更改成员角色。
- **移出领域** — 踢出该成员。
- **管理行政授权** — 打开该行政助理的**行政授权菜单**（仅限所有者）。

### 操作限制

| 操作 | 所有者 | 行政助理 |
|---|---|---|
| 提升 / 降级成员 | ✔ | ✔（仅限普通成员） |
| 移出成员 | ✔ | ✔（仅限普通成员） |
| 提升为行政助理 | ✔ | ✗ |
| 降级 / 移出其他行政助理 | ✔ | ✗ |
| 降级 / 移出所有者 | ✗ | ✗ |

> **注意：** 最多可设置 **3 名**行政助理。

### 行政授权

**所有者**可以为每位行政助理单独切换各项授权。在成员详情菜单中打开的**行政授权菜单**内进行配置。

所有行政授权**默认全部启用**。所有者可禁用其中任意项。

| 授权 | 说明 |
|---|---|
| **重命名领域** | 允许行政助理修改聚落总名称及各辖区范围名称 |
| **管理成员** | 允许行政助理提升 / 降级 / 移出成员 |
| **审核新成员申请** | 允许行政助理接受或拒绝申请者 |
| **编辑内政公告** | 允许行政助理创建和删除公告 |
| **国策树规划** | 允许行政助理管理国策树（计划中） |
| **地理范围修改** | 允许行政助理修改领地形状和大小 |
| **地块设置项修改** | 允许行政助理调整领地权限和规则 |
| **管理传送点** | 允许行政助理设置和配置传送点 |
| **移民政策** | 允许行政助理更改领域移民政策 |

## 审核新成员申请

点击**审核新成员申请**将打开**待审移民列表菜单**，列出所有处于**待审移民**状态的玩家。

- 点击申请者可打开其**审核菜单**，内含**接受**和**拒绝**按钮。

### 接受

- 申请者晋升为**成员**。
- 他们收到邮件通知。
- 若申请者为**受邀**玩家：加入费用从领域**国库**中扣除。若国库余额不足，审核将被阻止。
- 若申请者通过**申请制**政策直接申请：费用已在申请时由玩家支付。

### 拒绝

- 对于**直接申请**的玩家：加入费用**退还**至申请者账户，其状态设为**被拒移民**。
- 对于**受邀**申请者：邀请记录被删除；不涉及费用（邀请时未收取费用）。

> **注意：** 需要**审核新成员申请**行政授权。

## 内政公告管理

点击**编辑内政公告**将以管理员视图打开**公告列表菜单**，显示包括已删除（单独标记）在内的所有公告。

### 创建公告

在列表菜单中点击**创建公告**，铁砧界面出现 — 输入公告内容。

- 公告立即发布给所有成员。
- 成员下次登录或打开公告列表时会看到通知。
- 已读状态按成员单独追踪。

**对应命令：** `/community announcement create <communityIdentifier> <content>`

### 删除公告

在列表中点击公告以打开详情视图，然后点击**删除**。公告为软删除（对成员隐藏但保留在记录中）。

> **注意：** 需要**编辑内政公告**行政授权。

**对应命令：** `/community announcement delete <communityIdentifier> <announcementId>`

## 移民政策

**移民政策**按钮（槽位 28）每次点击循环切换三种政策：

| 政策 | 图标 | 行为 |
|---|---|---|
| **公开** | 绿色羊毛 | 玩家支付加入费用后立即成为成员 |
| **申请制** | 黄色羊毛 | 玩家支付费用，成为待审移民，等待行政助理审批 |
| **仅限邀请** | 红色羊毛 | 玩家只能通过成员邀请加入 |

> **注意：** 需要**移民政策**行政授权。
