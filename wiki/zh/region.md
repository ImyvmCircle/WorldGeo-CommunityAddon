每个领域都与一块**地块**（region）绑定 — 即 Minecraft 世界中一块定义好的地理区域。一个地块可以包含一个或多个**辖区范围**（scope），每个辖区范围可拥有独立的几何形状、权限设置和传送点。

## 辖区范围

**辖区范围**是领域地块内的一个命名子区域。主辖区（最外层）在领域创建时自动生成。可以在之后添加更多辖区范围，以实现更精细的控制。

### 添加辖区范围

要添加新的辖区范围，可通过以下两个入口：

- **圈地菜单**（主菜单槽位 13 的**圈地**按钮）→ **增加辖区**；或
- 内政管理菜单 → **地理范围修改** → **全局** → **添加辖区范围**。

两种方式均打开**辖区创建界面**（内置圈地模式切换、形状切换、命名，以及≥2个点位时显示的确认按钮）。需支付基础费用：

| 类型 | 辖区范围添加基础费用 |
|---|---|
| 庄园 | $500.00 |
| 领地 | $1,000.00 |

辖区范围还会针对其占用的领地按面积计价。选点后，系统生成待确认请求。

**对应命令：**
- `/community confirm_modification <regionId> <scopeName>` — 确认几何修改
- `/community cancel_modification <regionId> <scopeName>` — 取消几何修改

## 地理范围修改

内政管理菜单中的**地理范围修改**按钮（槽位 19，地图图标）打开**辖区选择菜单**。从列表中点击辖区后，将**开启修改现有辖区圈地模式**并关闭菜单，提示玩家在世界中选择新的边界点位。选点完成后，再次打开圈地菜单 → **修改地块**以确认。

也可直接通过主菜单**圈地**按钮（槽位 13）→ **修改地块**完整操作。

### 修改边界

**通过圈地菜单：**

1. 点击圈地菜单中的**修改地块**；
2. 若不在圈地模式中，选择聚落（若加入了多个），再点击辖区——开启**修改现有辖区**模式并关闭菜单；
3. 手持**命令方块**在世界中右键点击方块以设置新边界点位；
4. 重新打开圈地菜单 → **修改地块**——直接执行修改；
5. 聊天框出现费用摘要及可点击的**[确认]** / **[取消]**提示（有效期 5 分钟）。

**通过内政管理菜单：**

1. 打开内政管理菜单 → **地理范围修改** → 选择辖区；
2. 修改现有辖区圈地模式开启，菜单关闭并提示；
3. 按上述第 3–5 步操作。

**费用：** 扩大辖区范围需支付面积价格差额。缩小辖区范围退还费用差额的 **50%**。

> **注意：** 需要**地理范围修改**行政授权。

---

### 增加辖区范围

**通过圈地菜单：**

1. 点击圈地菜单中的**增加辖区**；
2. 选择聚落（若加入了多个），检查权限；
3. 打开**辖区创建界面**，内置圈地模式切换和形状控制；
4. 开启圈地模式，选择点位，然后命名并确认辖区。

**通过内政管理菜单：**

1. 打开内政管理菜单 → **地理范围修改** → **全局** → **添加辖区范围**；
2. 流程与上述相同。

**费用：** 辖区添加基础费用 + 面积计费（与聚落创建定价结构相同）。

## 地块设置项修改

内政管理菜单中的**地块设置项修改**按钮（槽位 20，海洋之心图标）打开**辖区选择菜单**。选择辖区以打开其**设置菜单**，在其中配置权限和规则。

### 权限

权限控制非所有者玩家在辖区范围内可以进行的操作。每项权限可在辖区范围内全局切换，或按玩家单独分配以实现更精细的控制。

| 权限名称 | 说明 |
|---|---|
| **建造/破坏** | 放置和破坏方块 |
| **建造** | 仅放置方块 |
| **破坏** | 仅破坏方块 |
| **访问容器** | 打开箱子、木桶、潜影盒等 |
| **交互** | 使用按钮、拉杆、门等 |
| **红石使用** | 与红石组件交互 |
| **村民交易** | 与村民交易 |
| **放置桶** | 用桶放置液体 |
| **取水桶** | 用桶舀取液体 |
| **耕作** | 耕耘土地、收割作物 |
| **PvP战斗** | 对其他玩家造成伤害 |
| **击杀动物** | 伤害被动动物 |
| **击杀村民** | 伤害村民 |
| **使用投掷物** | 投掷物品（雪球、鸡蛋、药水等） |
| **投掷鸡蛋** | 投掷鸡蛋 |
| **投掷雪球** | 投掷雪球 |
| **投掷药水** | 投掷 / 使用药水 |
| **点火** | 使用打火石或火焰弹 |
| **使用盔甲架** | 与盔甲架交互 |
| **使用物品展示框** | 与物品展示框交互 |

每项权限启用时按面积计费。恢复默认设置退还该费用的 **50%**。按玩家单独设置的费用为辖区范围全局设置费用的 **1/5**。

### 规则

规则控制辖区范围内的自动行为（不与特定玩家操作绑定）。

| 规则名称 | 说明 |
|---|---|
| **怪物生成** | 允许敌对生物在辖区内生成 |
| **幻翼生成** | 允许幻翼在辖区内生成 |
| **TNT方块保护** | 防止 TNT 破坏辖区内方块 |

规则启用时同样按面积计费。

> **注意：** 需要**地块设置项修改**行政授权。

**对应命令：**
- `/community confirm_setting <regionId>` — 确认设置修改
- `/community cancel_setting <regionId>` — 取消设置修改
