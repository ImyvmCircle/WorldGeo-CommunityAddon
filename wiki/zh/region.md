---

每一个聚落都占据着一块地理疆域。这块总疆域称为**地块**，承载聚落的边界、传送点、权限和规则。

一个聚落的版图不一定是一整块连续土地。祖地、新拓之地、飞地或特区可以分开存在，每一片独立区域都称为**辖区范围**。

一个聚落的实际疆域等于所有辖区范围覆盖起来的总和。聚落建立时自动拥有第一个主辖区，后续可以继续添加、出售或重画其他辖区。

主世界、地狱、末路之地的面积分别记录，形成跨维度的疆域账册。

---

## 一个聚落的疆域是如何构成的

聚落建立之初，系统会自动创建第一个辖区范围，也就是聚落的**主辖区**。这片土地是聚落最初的根基。

随着聚落发展，行政成员可以**增加新的辖区范围**来拓展版图。新增辖区可以紧邻主辖区，也可以成为分散飞地。每个辖区范围都可以独立设定形状，以贴合地形或建筑规划。

行政成员也可以重新划定某个辖区范围的边界。

所有辖区范围共同构成聚落在世界中实际掌控的土地。

## 圈地模式

要定义或修改一个辖区范围的边界，需要使用**圈地模式**在世界中选取点位。

- **进入圈地模式**后，手持**下界之星**，在世界中**右键点击方块**即可放置选择点位。**左键点击**下界之星可撤销（移除）最后一个已放置的点位（逐步后退）。
- 系统支持三种圈地形状：
  - **矩形**：选择两个对角顶点，自动生成矩形区域。
  - **圆形**：先选择圆心，再选择圆边上的一点，以确定半径。
  - **多边形**：依次选择所有顶点，围成自定义多边形（须为凸多边形）。
- 不必一次性完成所有点位的选择，可以退出菜单后随时继续。
- 建议用方块在地面标记边界点，方便日后参考。

> **提示：** 圈地模式有两种形态，对应不同的操作。**新建圈地模式**用于创建新的辖区范围，**修改圈地模式**用于调整已有辖区范围的边界。两者不能同时启用。如果玩家打开"修改地块"时收到警告，说明当前处于新建圈地模式，需要先在辖区创建菜单中关闭圈地模式，再修改地块。

## 增加辖区范围

新增辖区有两个入口：

- **圈地菜单**（主菜单 → **圈地**按钮）→ **增加辖区**；或
- 内政管理菜单 → **地理范围修改** → **全局** → **添加辖区范围**。

两种方式均会打开**辖区创建界面**。该界面提供以下操作：

1. **开启圈地模式**，进入世界中选择边界点位。
   - 菜单标题会提示当前步骤（"先开启圈地模式" → "在世界中选择点位"）。
   - 聊天框会先标出当前创建维度。主世界为绿色，地狱为红色，末路之地为偏金白高亮。
   - 点位充足后，会出现**命名**按钮和**确认创建**按钮。
2. **切换形状**，可在矩形、多边形和圆形之间循环。
3. 选完点位后**退出菜单**，在世界中继续选点；或直接**关闭圈地模式**（取消本次操作）。
4. 若想清空已选点位重新来过，点击**重置选点**按钮。
5. 凑够点位并命名后，点击**确认创建**，系统生成费用摘要并等待聊天框确认。

创建确认消息会继续显示新辖区所在的维度，让这片新增疆域从一开始就与所属世界对应起来。

创建一个新辖区范围的费用如下。

| 聚落类型 | 新增辖区基础费用 |
|---|---|
| 庄园 | $500.00 |
| 领地 | $1,000.00 |

此外，还需按辖区范围所覆盖的面积额外计费。

若新辖区位于地狱，直接绑定该辖区的费用按 ×8 结算。位于末路之地时按 ×2 结算。主世界保持 ×1。聚落总地价与总设置项费用，则按维度分别计算后汇总。

**软性上限机制：** 推荐的辖区数量上限为 `ceil(正式成员数 / 2)`（正式成员指所有者、行政助理或成员；例如 1 名成员对应上限 1，2 名成员对应上限 1，3 名成员对应上限 2）。超过此上限**不会阻止**继续创建，但**创建总费用（基础费用 + 面积费用 + 设置费用）**将整体乘以 **1.5^N**，其中 N 为超出上限的数量。例如上限为 2、已有 2 个辖区时，再创建第 3 个（N=1），总创建费用乘以 ×1.5；再创建第 4 个（N=2）则乘以 ×2.25。该附加费仅对新建辖区的创建费用生效，不影响已有辖区修改边界时的面积差价收费或退款。超出上限时，系统会在确认前发出提示通知，确认界面会显示完整公式，最终扣费消息也会标注超限附加费金额。

> **注意：** 只有处于**激活状态**的聚落才能添加辖区范围。招募中或待审核阶段的聚落尚不允许此操作，需等待通过 OP 审核进入激活状态后方可进行。

创建确认和取消都通过聊天框按钮完成。

> **名称引号：** 命令中的 `<scopeName>`（以及其他地方用到的领域/聚落名称）若包含 ASCII 字母和数字以外的字符，**必须用英文双引号括起来**，例如 `"示例辖区"`。按 Tab 键自动补全时会自动添加引号。

## 出售辖区范围

行政成员可以将某个辖区范围出售给系统。系统根据该辖区的面积和受影响的设置项计算退款金额，退还相应费用差额的 **50%**，不退还创建时的基础费用。每个聚落必须至少保留一个辖区范围，因此最后一个辖区无法出售。

**操作步骤**

1. 内政管理菜单 → **地理范围修改** → **全局** → **出售辖区范围**；
2. 从列表中选择要出售的辖区范围；
3. 聊天框出现退款摘要及 **[确认]** / **[取消]** 提示（5 分钟内有效）。

> **注意：** 只有处于**激活状态**的聚落才能出售辖区范围。

> **注意：** 需要**地理范围修改**行政授权。

出售确认和取消都通过聊天框按钮完成。

---

## 修改辖区范围边界

行政成员可以重新划定某个已有辖区范围的边界。修改前，系统会比较修改前后的总面积，扩大时从国库收费，缩小时把应退金额记到所有者的个人贡献记录中。

**通过圈地菜单（推荐）：**

1. 主菜单 → **圈地** → **修改地块**；
2. 若玩家尚未进入任何圈地模式，选择聚落（加入多个聚落时），再从列表中点击要修改的辖区范围；
3. 系统自动进入**修改圈地模式**，菜单关闭，并提示玩家在世界中选取新的边界点位；
4. 手持**下界之星**，右键点击方块设定新边界（左键点击可撤销上一个点位）；
5. 再次打开圈地菜单 → **修改地块**，确认修改；
6. 聊天框出现费用摘要及**[确认]** / **[取消]**提示（5 分钟内有效）。

**通过内政管理菜单：**

1. 内政管理菜单 → **地理范围修改** → 从列表选择要修改的辖区范围；
2. 进入修改圈地模式，菜单关闭并提示；
3. 按上述第 4–6 步操作。

无论通过哪一种入口，发起修改与进入修改确认时，玩家都必须身处**目标辖区所在维度**。若玩家在别的维度点选该辖区，菜单会关闭，聊天框会直接说明目标辖区所在维度与当前位置，并附带一个可点击的**返回界面**按钮。

> **视觉反馈：** 进入修改圈地模式后，系统会以**橙色粒子**描绘现有辖区范围的完整边界轮廓，方便玩家在选取新点位时看到即将被替换的旧形状。

**费用规则：**
- 扩大辖区面积：支付新增面积的差价。
- 缩小辖区面积：退还缩减面积差价的 **50%**。

确认界面会列出修改前后各维度面积，并额外标示发生变化的维度及增减量。

> **注意：** 只有处于**激活状态**的聚落才能修改辖区范围边界。招募中或待审核阶段的聚落尚不允许此操作。

> **注意：** 需要**地理范围修改**行政授权。

## 地块设置项修改

每个辖区范围可以独立配置**权限**与**规则**，整个聚落也可以设置全局规则。权限决定玩家能否主动操作，规则决定区域里的自动行为。

进入方式：内政管理菜单 → **地块设置项修改** → 选择要配置的辖区范围 → **设置菜单**。

### 权限

权限控制玩家在辖区范围内可以进行的操作。每项权限可对整个聚落或某个辖区生效，也可对单个玩家单独设置。

| 权限名称 | 说明 |
|---|---|
| **建造/破坏** | 放置和破坏方块 |
| **建造** | 仅放置方块 |
| **破坏** | 仅破坏方块 |
| **访问容器** | 打开箱子、木桶、潜影盒等 |
| **交互** | 使用按钮、拉杆、门等 |
| **红石使用** | 与红石组件交互 |
| **村民交易** | 与村民交易 |
| **放置桶** | 用桶放置液体 |
| **取水桶** | 用桶舀取液体 |
| **耕作** | 耕耘土地、收割作物 |
| **PvP战斗** | 对其他玩家造成伤害 |
| **击杀动物** | 伤害被动动物 |
| **击杀村民** | 伤害村民 |
| **使用投掷物** | 投掷物品（雪球、鸡蛋、药水等） |
| **投掷鸡蛋** | 投掷鸡蛋 |
| **投掷雪球** | 投掷雪球 |
| **投掷药水** | 投掷 / 使用药水 |
| **点火** | 使用打火石或火焰弹 |
| **使用盔甲架** | 与盔甲架交互 |
| **使用物品展示框** | 与物品展示框交互 |
| **投掷旋风弹** | 投掷旋风弹 |

每项权限改成与默认状态不同的值时按面积计费，恢复默认退还该费用的 **50%**。按玩家单独设置的费用为全局费用的 **1/5**。

挂在整个聚落上的权限，按各维度面积分别计价后汇总。挂在单个辖区范围上的权限，继承该辖区的维度倍率。

当整个聚落的全局权限被改成与默认状态不同，并且该权限不是 PvP 或飞行时，系统会尽量给已有正式成员补上个人默认权限。新成员加入时也会触发同样的补授权逻辑，避免聚落对外限制误伤正式成员。

### 规则

规则控制辖区范围内的自动行为（不与特定玩家操作绑定）。

| 规则名称 | 说明 |
|---|---|
| **怪物生成** | 允许敌对生物在辖区内生成 |
| **幻翼生成** | 允许幻翼在辖区内生成 |
| **TNT方块保护** | 防止 TNT 破坏辖区内方块 |
| **末影人捡方块** | 允许末影人在辖区内拾取方块 |
| **幽匿扩散** | 允许幽匿感测体在辖区内扩散 |
| **雪人雪迹** | 允许雪傀儡在辖区内留下雪迹 |
| **发射器射击** | 允许发射器在辖区内发射物品 |
| **压力板触发** | 允许辖区内的压力板被触发 |
| **活塞移动** | 允许活塞在辖区内推动方块 |

规则改成与默认状态不同的值时同样按面积计费，恢复默认退还 **50%**。

设置确认消息会列出当前各维度面积。
