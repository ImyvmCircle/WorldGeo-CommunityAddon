**领域**（Community）是 IMYVM 服务器上与专属地理区域绑定的玩家组织。领域让玩家能够共同管理领土、私密通信、建立集体经济，并在结构化成员体系下协作。

运行 `/community` 即可打开**社群主菜单**——所有领域功能的起点。每项功能均可通过箱子式 GUI 菜单访问；命令是进行相同操作的替代入口。

## 领域类型

|  | 庄园 | 领地 |
|---|---|---|
| 规模 | 小型 | 大型 |
| 基础创建费用 | $5,000.00 | $10,000.00 |
| 成员上限 | 4（默认） | 无限制 |
| 激活所需最少成员数 | — | 4 |
| 加入费用 | $1,500.00 | $500.00 |
| 激活路径 | 服务器管理员审批 | 招募 4+ 原始股玩家 → 服务器管理员审批 |

## 领域状态

| 状态 | 适用类型 | 说明 |
|---|---|---|
| **招募中** | 仅限领地 | 正在寻找原始股玩家的领域；行政权限受限 |
| **审核中** | 两者均有 | 等待服务器管理员审批；行政权限受限 |
| **活跃** | 两者均有 | 活跃；所有功能解锁 |
| **已撤销** | 两者均有 | 已停用；所有操作被封锁 |

### 状态流转路径

- **庄园：** 创建 → **审核中** → （服务器管理员审批）→ **活跃** / **已撤销**
- **领地：** 创建 → **招募中** → （48小时内招募4+名原始股玩家后自动进入）→ **审核中** → （服务器管理员审批）→ **活跃** / **已撤销**

> **注意：** 领地若在 48 小时内未能招募到足够的原始股玩家，将自动被撤销并退还创建费用。

## 成员角色

| 角色 | 庄园显示名称 | 领地显示名称 | 说明 |
|---|---|---|---|
| **所有者** | 地主 | 领主 | 拥有完整权限；无法退出领域 |
| **行政助理** | 管家 | 执事 | 受委托的管理权限（受所有者配置的行政授权限制） |
| **成员** | 住户 | 居民 | 普通成员；可捐赠、查看、聊天、邀请、退出 |
| **待审移民** | — | — | 等待成员资格审核 |
| **被拒移民** | — | — | 已被拒绝的申请者 |

> **注意：** 最多可设置 **3 名**行政助理。
