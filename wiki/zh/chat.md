每个领域都有一个**内部频道**，仅对正式成员（所有者、行政助理、成员）可见。成员可以发送和阅读领域内部消息，这些消息不会显示在全局聊天中。

## 打开聊天室

在[领域菜单](community-menu.md)中点击**内部频道**（槽位 28，可写书图标）以打开**聊天室菜单**。

### 聊天室菜单

| 按钮 | 说明 |
|---|---|
| **切换内部频道** | 启用或禁用该领域的内部频道作为默认聊天目标 |
| **查看聊天记录** | 浏览该领域的历史消息 |
| **如何发送消息** | 显示使用说明 |

菜单还显示该领域的当前**频道状态**：

| 状态 | 含义 |
|---|---|
| **已激活**（绿色） | 你输入的消息将发送到该领域的频道 |
| **未激活**（灰色） | 你的消息将发送到全局聊天 |
| **其他频道**（黄色） | 你有另一个领域的频道处于激活状态 |

## 发送消息

### 方法一 — 通过频道切换（推荐）

1. 打开[领域菜单](community-menu.md) → **内部频道**。
2. 点击**切换内部频道**，将该领域设为你的活跃频道。
3. 关闭菜单，在聊天栏正常输入 — 消息将发送到领域内部频道。
4. 要返回全局聊天，再次打开聊天室菜单并点击**切换内部频道**以停用。

> **注意：** 同一时间只能有一个领域频道处于激活状态。启用新频道将自动停用之前的频道。

### 方法二 — 通过命令

无需更改活跃频道，发送一次性消息：

`/community chat <communityIdentifier> <message>`

或使用快捷别名：

`/ch <communityIdentifier> <message>`

> **注意：** 领域名称含空格时须用双引号包裹，例如 `/ch "我的领域" 你好！`

### 通过命令切换活跃频道

`/community chat_channel <communityIdentifier>`

## 聊天记录

在聊天室菜单中点击**查看聊天记录**（或运行对应命令）将在聊天框中显示领域频道的最后 **20 条消息**，包括发送者姓名、角色标签和时间戳。

> **注意：** 聊天记录中的角色标签在代码中以英文硬编码，不随语言设置变化：

| 角色 | 聊天显示标签（英文硬编码） |
|---|---|
| 所有者（庄园）地主 | `Landowner` |
| 所有者（领地）领主 | `Lord` |
| 行政助理（庄园）管家 | `HouseKeeper` |
| 行政助理（领地）执事 | `Steward` |
| 成员（庄园）住户 | `Resident` |
| 成员（领地）居民 | `Citizen` |

---

## 访问权限

只有**正式成员**（所有者、行政助理、成员）可以发送或阅读领域聊天消息。**待审移民**和**被拒移民**无法使用内部频道。
